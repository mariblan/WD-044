Instructions

  1. One of the team from the psd project is going to create a branch with the name of the team and add a folder in the git&gihub-workshop with the name of the group.
  2. The other team members have to pull this branch in the computer.
  3. From this branch each one is going to create a new individual branch name feature/name-of-the-feature.
  4. Each team member should do three commits and pushes into the repo.
  5. When the code is done each person has to create a pull request for their branch
  6. Three people has to approve the pull requests. You can leave messages of things to improve before approving.
  7. When all is done, the branches should be merge with the branch with the name of the group.
