Instructions for approvals in pull requests

  1. The person who creates the pull request adds the collaborators on the Reviewers.
  2. The Colaborators go to the Files changed and they can do the review of the changes there and then approve.
  3. Inside the Files changed there is the blue button Review changes. Click on it to see the options.
  4. Comment let's you leave feedback without approving or forcing the person to do any actual changes.
  5. Approve will automatically approve the pull request as it is.
  6. Request changes will let you give feedback that forces the person who created the pull request to change the parts the reviewr specified in the feedback.
  7. Once you have selected one of the options, click on submit review.
